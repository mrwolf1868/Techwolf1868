# TECHWIZARD WhatsApp Bot

A powerful, feature-rich WhatsApp bot with Gemini AI integration.

## Features
- AI Chat (via `.ai`)
- VCF Contact Export
- Pairing Code Dashboard
- Always Online presence
- Group Management

## Deployment
Click the **Deploy** button in AI Studio and select **Cloud Run**.
Ensure `GEMINI_API_KEY` is set in your environment variables.
